About This Blog

Welcome to Embedded Essentials, a technical blog dedicated to exploring the fascinating world of Linux kernel development. Whether you're a beginner looking to make your first contribution or an experienced developer diving into kernel internals, you'll find detailed guides, tutorials, and insights here.

This blog covers everything from setting up your development environment to understanding complex subsystems like Media, DRM & KMS, ALSA etc. and device drivers. Each article is written with clarity and depth, backed by practical examples and real-world scenarios.

#include <linux/module.h>
#include <linux/kernel.h>

static int __init hello_init(void)
{
    printk(KERN_INFO "Welcome to Embedded Essentials !\n");
    return 0;
}

static void __exit hello_exit(void)
{
    printk(KERN_INFO "Thanks for visiting!\n");
}

module_init(hello_init);
module_exit(hello_exit);

MODULE_LICENSE("GPL");
MODULE_AUTHOR("Shrikant Raskar");
