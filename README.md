# Embedded Essentials Blog

A modern, lightweight blog focused on Linux kernel development, system programming, and embedded systems.

URL: [embedded-essentials.github.io](https://embedded-essentials.github.io/)

---

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/embedded-essentials/embedded-essentials.github.io.git
   cd embedded-essentials.github.io
   ```

2. Start a local server:
   ```bash
   python3 -m http.server 8080
   ```

3. Open browser:
   ```
   http://localhost:8080
   ```
